# Sigma Toolkit

Projeto com diversas ferramentas para identificação e ações que lidam com cadeias e linguagens.

## Build

1. Instale o Dotnet 9.0
2. Rode o programa digitando no terminal:
```sh
dotnet run --project SigmaToolkit
```
